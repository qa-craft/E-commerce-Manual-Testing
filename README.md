# E-commerce Manual Testing Project

This repository contains a **manual testing project for an E-commerce web application.**

The goal of this project is to demonstrate real-world **software testing documentation and QA workflow** including requirement analysis, test planning, test design, execution, and bug reporting.

---

# Application Tested

E-commerce Web Application

Core user flow tested:

* User Login
* Product Search
* Add to Cart
* Cart Management

---

# Modules Covered

1. Login Module
2. Product Search Module
3. Cart Module

---

# Testing Types Performed

* Functional Testing
* UI Testing
* Negative Testing
* Boundary Testing

---

# Testing Artifacts Included

This project contains the following QA documents:

* Requirements
* Test Plan
* Test Scenarios
* Test Cases
* Test Execution Report
* Bug Report Log
* Requirement Traceability Matrix (RTM)

All documents are available in the **Test_Documents** folder.
